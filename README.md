# Projetos em JQuery e Ajax

This repository consolidates my knowledge and practical experience with jQuery and Ajax, gained through a Udemy course and additional research conducted independently. 

It serves as a personal learning milestone and a reference for future projects.

<hr>

## Destaque 

The repository features the App Dashboard project, a fully functional web application built with jQuery with an database integration.
